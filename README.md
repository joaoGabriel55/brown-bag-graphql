# GraphQL using Apollo Server/Client - Codeminer42 Brown Bag

## How to run locally?

### Backend

- Setup `.env` file, server port and db url
- Run `yarn run dev`

### Frontend

- Check and setup `.env` file;
- Run `yarn start`
